# Sysdig
**Sysdig** is a universal system visibility tool with native support for containers:  
`~$ sysdig`

**Csysdig** is a simple, intuitive, and fully customizable curses UI for sysdig:  
`~$ csysdig`

Additional information available [here](https://github.com/draios/sysdig).

>**Note:** Artifacts available in this repository are targeted for `s390x` architecture.
